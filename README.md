
# 🎮 Gamified Task System（个人游戏化任务系统）

一个用于提升执行力的个人工具，通过“任务 + 随机奖励 + 积分兑换”机制，让枯燥任务变得像游戏一样有反馈。

---

## ✨ Features（功能）

最小实例：
- 添加任务获得随机的积累奖励
- 执行任务时根据显示时间秒钟，每10秒累积奖励可能得，可能不得
    - 任务可以开始，暂停，保存任务的开始时间，暂停时间，完成时间
- 完成任务获得奖励
- 显示奖励：金币，和钻石

进一步：
- 获得的奖励作为自走棋游戏的奖励
- 获得的奖励作为养宠物游戏的奖励
- 一周进行排位结算

---

## 🧠 Motivation（为什么做）

传统 ToDo 工具的问题：

* 没有即时反馈
* 缺乏奖励机制
* 很容易放弃

本项目尝试用游戏化机制解决：

> “如何让自己更愿意开始任务，并坚持完成？”
